# extract_snp_test_barcode

## 描述

提取出5个有较多突变（尽量包含T790M和L858R突变）的细胞的barcode

## 使用

输入文件

- snp分析结果中的`<sample_name>__count_tsne.tsv`，从这个文件中得到CID和barcode的对应关系以及每个barcode的突变数量
